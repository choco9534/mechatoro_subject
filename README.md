# mechatoro_subject

<<メカトロ実習のプログラムを置いておく場所>>

#置いてあるもの
---
・最初の綺麗なプログラム
・ロボ本体を動かすプログラム
・機構を動かすプログラム

#ロボ本体に必要な機能
---
・オブジェクトを見つける
・オブジェクトに近づく
・いい感じの距離で止まる
　・場外に出ない
・つかむ
・持ち上げる
・離す
・最初から繰り返す

#使うもの
・Arduino UNO
・SG90
・SG90-HV x 2
・DCモーター x 2
・スイッチ付き赤外線センサー
・超音波センサー

#改善点
---
・動きが速くて小さいオブジェクトを見つけられない気がする
  ┗ゆっくり走らせたい
